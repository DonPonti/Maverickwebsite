Right now every listing on this site is placeholder data, dropped in to show the
shape of the directory. The real version will be built from founder submissions,
a short form, and some manual verification before anything goes live.

If you'd rather not wait, the listings live in a single data file — see the
`_data/startups.json` file in the project — so swapping in real companies is a
matter of editing that file, not the templates.
